+  心得与收获： 
   +  防抖节流
      + 利用闭包和定时器控制函数执行的频率
      + timer变量一直可以被访问修改，是因为通过闭包让它一直存在没被销毁
      +  非立即执行模式等待delay时间内不再触发,连续不断触发会不断重置计数器
         + 场景在搜索框输入联想，窗口resize
         + 滚动停止后计算位置
      +  立即执行
         + 触发时就立刻执行，delay时屏蔽所有触发，只执行第一次
         + 先执行，冷却期间不等你
         + 按钮防误触发，表单重复提交
   +  深拷贝：
      +  WeakMap优点就是弱引用，在循环引用时有助于垃圾回收，防止内存泄露
      +  JSON.parse(JSON.stringify(obj))
         + undeifned/function/Symbol会丢失
         + Date会变成字符串,无法处理循环引用
      + map.set在for...in位置之前的原因是，在递归循环引用属性时，可以从map中找到一创建的属性
