+  心得与收获(总结学习成果，要详细说明对相关知识点的个人理解。):
   +  html
      +  html标签是为了帮助文本，图片以及实现操作功能渲染到浏览器
      +  有语言和编码规范,语义化直观
      +  input的类型多样可以实现多种功能选择
      +  资源调用
         +  通过相对定位和绝对定位实现调用文档资源
   +  CSS
      +  层叠性对同一个标签的属性后书写的可以覆盖先书写，方便实现父子元素的重写
      +  继承性
         +  子元素可以继承父元素的部分属性，减少属性代码书写，让结构更加清晰
      +  优先级
         +  内联样式 > id 选择器 > 类选择器 = 伪类选择器 = 属性选择器 > 标签选择器 = 伪元素选择器，优先级是有权重的权重可以比较不会叠加
         +  ！important例外
      +  伪类是基于元素为基准，对元素状态操作的动态操作，状态改变操作随之消失，
      +  伪元素是基于当前元素内容，是控制在元素上的，但是并不存在于文档,所以:after，：before之后，之前操作
      +  absolute绝对定位，和float浮动会脱离标准流,并且层级会上升，压住其他元素
      +  清除浮动使用空标签，父级元素添加overflow,伪元素设置content文本方式，清除浮动对后续元素的影响
      +  定位一般按照子绝父相，如果父元素没有就会依据html元素
      +  元素居中，块级使用margin padding,但需要设置宽度高度,行内元素文本居中使用text-align:center；定位居中会重叠元素，padding可以实现内容居中但是行高会随着内容增加变高,position和transfrom（-50%,-50%）可以实现垂直居中
