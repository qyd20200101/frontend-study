## 🎁 新增功能汇总

### 1. 组件类型扩展

| 新组件 | 类型 | 用途 | 配置 |
|--------|------|------|------|
| 🆕 **textarea** | 多行文本 | 描述、备注 | rows, maxLength, resize |
| 🆕 **number** | 数字输入 | 价格、年龄 | min, max, step |
| 🆕 **time** | 时间选择 | 工作时间 | format, isRange |

### 2. 验证规则新增 (10 个新类型)

| 验证类型 | 说明 | 示例 |
|---------|------|------|
| 🆕 **email** | 邮箱格式 | user@example.com |
| 🆕 **phone** | 手机号 | 13800138000 |
| 🆕 **url** | URL 格式 | https://example.com |
| 🆕 **pattern** | 正则表达式 | /^[0-9]+$/ |
| 🆕 **minLength** | 最小长度 | 至少 6 个字符 |
| 🆕 **maxLength** | 最大长度 | 最多 20 个字符 |
| 🆕 **min** | 最小值 | 大于等于 0 |
| 🆕 **max** | 最大值 | 小于等于 100 |
| 🆕 **custom** | 自定义验证 | 自定义函数 |
| 🆕 **预定义模板** | 快速应用 | getValidationTemplate() |

### 3. 工具函数新增 (8 个)

```typescript
✨ formatComponentProps(component)        // 格式化属性
✨ generateOptionsTemplate(component)    // 生成选项模板
✨ validateComponentConfig(component)    // 验证配置完整性
✨ cloneComponent(component)             // 克隆组件
✨ getInputType(type)                    // 获取 input type
✨ buildValidationRule(rule, comp)       // 构建验证规则
✨ getTriggerByType(type)               // 获取触发事件
✨ getValidationTemplate(type)          // 获取验证模板
```

### 4. 接口扩展 (新增字段)

```typescript
// FormComponent 新增字段
- validation?: ValidationRule[]   // 复杂验证规则
- disabled?: boolean              // 禁用字段
```

```typescript
// 多行文本
{ type: 'textarea', label: '备注', field: 'remarks' }

// 数字输入
{ type: 'number', label: '价格', field: 'price', props: { min: 0, max: 99999 } }

// 时间选择
{ type: 'time', label: '时间', field: 'time', props: { format: 'HH:mm' } }
```

### 使用验证规则

```typescript
// 方式一：快速应用模板
import { getValidationTemplate } from '@/utils/validation';
component.validation = getValidationTemplate('email');

// 方式二：自定义规则
component.validation = [
  { type: 'required', message: '必填' },
  { type: 'email', message: '邮箱格式错误' }
];
```
